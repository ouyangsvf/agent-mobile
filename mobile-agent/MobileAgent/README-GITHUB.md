## 下载 APK

在 [Releases](../../releases) 页面下载最新版本

## 快速开始

1. Fork 此仓库
2. 配置 GitHub Secrets（可选，用于签名）
3. Push 代码触发自动构建
4. 在 Actions 页面下载 APK

查看 [GITHUB_ACTIONS_SETUP.md](./GITHUB_ACTIONS_SETUP.md)
